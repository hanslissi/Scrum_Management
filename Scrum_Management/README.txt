 Here are some instructions on how to start my program:

 1.) You can find a backup file of the Database inside the root folder of the repository
 2.) Restore the backup in pgAdmin
 3.) Go to the dist folder
 4.) Open the preferences.txt file
 5.) Change the settings. ServerURL, User and password (There are sample inputs)
 6.) Start the .jar file in the dist folder.
 7.) Everything should work and some Users and Tasks should be loaded from the Database

 When something is not working please don't hesitate to contact: riejoc16@htlkaindorf.at
